# SynkkAfrica Admin Dashboard

A modern, scalable, and responsive **admin dashboard** for managing the **SynkAfrica platform**.  
Built with **Next.js**, **JavScript**, and **TailwindCSS**, the dashboard provides administrators and operations teams with tools to manage vendors, services, bookings, transactions, user accounts, and customer support efficiently.
